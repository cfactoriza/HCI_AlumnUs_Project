# HCI_AlumnUs_Project
HCI Project
This is a project done by Christian Factoriza, Andy Situ, Adam Inga, and Lucas Herrera.
AlumnUs is a place for alumni to help undergraduate students that are seeking guidance from people who have already graduated and are relatviely experinced ont what needs to be done for their respective majors.
This will help students be able to be more prepared and learn from alumni's experiences to influence their career decisions in a postive way.
